TIM Interface Box

2017

The Target Interface Module for ADSSS includes dense electrical connectors for its various I/O. In lab testing, the team frequently had to build bespoke cable harnesses to interface with the TIM or use risky approaches such as directly probing the connectors. To reduce risk and expedite lab testing, I designed and built an interface box that not only breaks out the I/O but also includes selectable input stimulus and output indicators for acceptance testing and troubleshooting.
